# Dual-Arduino Teleoperated Robotic Arm

## Summary

Built a master/slave robotic arm system using potentiometer inputs, I2C communication, and servo actuation for real-time teleoperation.

## My Role

Team project. Responsible for mechanical assembly, electrical integration, and control logic support.

## Engineering Work

- Used potentiometers to read operator input angles
- Sent control data between Arduino boards using I2C communication
- Converted input values into servo commands
- Assembled the mechanical structure, wiring, and actuation system

## Tools

Arduino, I2C, servo control, potentiometers, mechanical assembly

## Next Improvements

Future revisions would improve calibration, add smoother motion profiles, and strengthen the mechanical structure.
